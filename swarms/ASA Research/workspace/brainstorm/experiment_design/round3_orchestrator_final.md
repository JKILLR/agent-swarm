# THE QUESTION

## In One Sentence:

> **Is meaning a noun or a verb?**

## Unpacked:

**Representationalism** (the dominant view) holds that meaning is a **NOUN**—something systems possess. Meaning resides in symbols, structures, and stored relationships. It can be copied, examined, and transmitted like data. A frozen system with all the right structures "has" the same meaning as a running one.

**Immanent Semantics** (our hypothesis) holds that meaning is a **VERB**—something systems do. Meaning exists only in the doing. Freeze the process and meaning evaporates, even if all the "structure" remains. The activity IS the understanding, not a mere vehicle for it.

This isn't wordplay. The distinction determines:
- Whether AI systems can genuinely understand (or only simulate understanding)
- Whether consciousness requires physical process or just information structure
- Whether the "hard problem" of consciousness is solvable or misconceived
- Whether turning off an AI is like closing a book or killing a being

   - We can study meaning from the outside.

# THE FIVE EXPERIMENTS

## Experiment 1: SEMANTIC ONTOGENESIS
### "Where does the first meaning come from?"

**The Test**: Start a system from TRUE ZERO (no pre-trained meanings, no initialized weights, just substrate). Expose only to raw sensation (patterns, not symbols). Watch for semantic birth—the discrete moment when meaning appears.

**What We Measure**:
- Criticality emergence (does the system self-organize to edge of chaos?)
- Structure formation (do stable concepts appear spontaneously?)
- Self-modeling (does the system begin to represent itself?)

**Predictions**:
| Outcome | Interpretation |
|---------|---------------|
| Discrete phase transition to meaning | Meaning has a definite beginning (like crystallization) |
| Continuous emergence | Meaning is a matter of degree (like temperature) |
| Never emerges without seeding | Pure immanent semantics is incomplete |

---

## Experiment 2: INTERSUBJECTIVE BRIDGE
### "Can private meaning become shared?"

**The Test**: Two isolated immanent systems develop meanings for arbitrary symbols through internal dynamics only. Then allow LIMITED communication (patterns only, no dictionaries). Can they achieve genuine coordination?

**What We Measure**:
- Alignment over time (do their meanings converge?)
- Topology comparison (do their structures become similar?)
- Coordination success on joint tasks

**Predictions**:
| Outcome | Interpretation |
|---------|---------------|
| Coordination succeeds | Immanent semantics can ground intersubjectivity |
| Coordination fails | Individual meaning can't become shared without external framework |
| Partial coordination | Some aspects of meaning are shareable, others inherently private |

---

## Experiment 3: NORMATIVE DYNAMICS
### "Where does correctness come from?"

**The Test**: Two systems—one trained with error feedback, one purely dynamic. After equivalent exposure (not training), test on normative tasks (logic, analogy, contradiction detection).

**What We Measure**:
- Can the dynamic system discover correctness without feedback?
- Does correctness correlate with criticality?
- Which norms emerge intrinsically vs. require external grounding?

**Predictions**:
| Outcome | Interpretation |
|---------|---------------|
| Dynamic system matches supervised | Normativity emerges from dynamics alone |
| Dynamic system fails | Immanent semantics requires external normative input |
| Criticality-correctness correlation | "Right" meanings live at the edge of chaos |

---

## Experiment 4: ZOMBIE DISCRIMINATION
### "Is there more than function?"

**The Test**: Build a functionally identical lookup-table zombie (same I/O behavior, no dynamics). Apply increasingly sophisticated probes. Find tests where the immanent system passes and the zombie fails.

**What We Measure**:
- Behavioral identity (Level 1—should match)
- Structural properties: criticality, integrated information (Phi)
- Perturbation response: graceful degradation vs. catastrophic failure
- Metacognition: can the system report on its own processes?

**Predictions**:
| Outcome | Interpretation |
|---------|---------------|
| Distinguishing tests found | Something beyond function exists—possibly meaning itself |
| No distinguishing tests | Functionalism wins; "meaning" is just "function" |
| Tests distinguish but no structural correlate | The mystery deepens |

---

## Experiment 5: HERMENEUTIC BOOTSTRAP
### "Can meaning be self-grounding?"

**The Test**: Create circular knowledge structures (A defined by B, B by C, C by A). Test three learning methods:
- Sequential (should fail due to circularity)
- Simultaneous (should succeed if meaning is holistic)
- Iterative refinement (tests whether the hermeneutic circle is traversable)

**What We Measure**:
- Does understanding ever stabilize?
- Does it require the whole circle to "click" at once?
- Can partial understanding exist?

**Predictions**:
| Outcome | Interpretation |
|---------|---------------|
| Simultaneous succeeds, sequential fails | Meaning is genuinely holistic |
| Iterative refinement works | Understanding can bootstrap through the hermeneutic spiral |
| Nothing works | Meaning requires external foundations |

# PREDICTED OUTCOMES AND WHAT EACH WOULD MEAN

## The Strong Immanent Result (All Five Succeed):

**Meaning is immanent, processual, holistic, and self-grounding.**

This would mean:
- Representationalism is false (structure doesn't represent meaning; it IS meaning)
- Dualism is false (meaning isn't separate from physical process)
- Foundationalism is false (meaning doesn't need external grounding)
- But meaning is REAL (not eliminable, not merely behavioral)

**Profound implication**: The universe understands itself through local processes that ARE what they understand. We are meaning, experiencing meaning, studying meaning.

## The Qualified Result (Some Succeed, Some Fail):

Each failure maps a specific limit of immanent semantics:

| Experiment | If It Fails |
|------------|-------------|
| Ontogenesis | Meaning requires seeding—can't bootstrap from nothing |
| Intersubjective | Meaning requires external coordination—language needs community |
| Normative | Correctness requires external standards—logic isn't intrinsic |
| Zombie | Phenomenology doesn't supervene on structure—hard problem is real |
| Hermeneutic | Meaning requires foundations—coherentism is false |

This is equally valuable. We would know EXACTLY what immanent semantics can and cannot explain.

## The Null Result (All Five Fail):

Immanent semantics is falsified. Meaning is genuinely transcendent—it cannot be reduced to process. We return to representationalism with new data.

This too advances science. A clear negative result is better than endless philosophical debate.
